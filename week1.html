Variabelen (opslag voor gegevens)

let naam = "Jamal"; // kan je later veranderen
const leeftijd = 17; // kan NIET meer veranderen
Wanneer gebruik je wat?
  • let ‚Üí waarde kan veranderen (bijv. score in game)
  • const ‚Üí waarde blijft hetzelfde (bijv. je naam)

Datatypes (soorten gegevens)

let getal = 10; // number (getallen)
let tekst = "Hallo"; // string (tekst, altijd tussen "")
let klopt = true; // boolean (true of false)
LET OP het verschil:
10 + 10 // = 20 (rekenen met getallen)
"10" + "10" // = "1010" (plakken van tekst)

Output (laten zien aan gebruiker)

In de console:

Pop-up melding:

In HTML schrijven:

HTML element veranderen:

Input (vragen aan gebruiker)

// Vraag stellen (pop-up)
prompt("Wat is je naam?");

// Antwoord opslaan in variabele
let naam = prompt("Wat is je naam?");

// Antwoord tonen op pagina
document.write("Hallo " + naam);

DOM selecteren (HTML aanpassen)

Wat betekent het?
  • document = je index.html bestand
  • getElementById("titel") = zoek een element met id="titel"
HTML:
<p id="begroeting">Hallo</p>
JavaScript:
document.getElementById("begroeting").innerHTML = "Hoi!";
Onthoud: getElementById zoekt het element met die id op

Teksten samenvoegen

let voornaam = "Ali";
let achternaam = "Bakker";

// Met + plak je teksten aan elkaar
let volledigeNaam = voornaam + " " + achternaam;
// Resultaat: "Ali Bakker"

document.write("Mijn naam is " + volledigeNaam + "!");
// Resultaat: "Mijn naam is Ali Bakker!"
