Week 2 - Functies en onclick

Functies (herbruikbare code)

Wat is een functie?
Een functie is een stukje code dat je kunt hergebruiken. Je schrijft het één keer en gebruikt het zo vaak je wilt.

Functie maken:

function zegHallo() {
  console.log("Hallo!");
}

Functie aanroepen:

zegHallo();
// Nu gebeurt het pas!
Onthoud: Functie maken = recept opschrijven, aanroepen = recept gebruiken

document.getElementById uitleg

Wat betekent het?
  • document = je index.html bestand
  • getElementById("titel") = zoek een element met id="titel"
Stapje voor stapje:
document.getElementById("titel")
1. document → kijk in je HTML
2. get → pak
3. ElementById → een element met een bepaalde id
4. ("titel") → namelijk id="titel"

Voorbeeld:

HTML:
<h1 id="titel">Welkom</h1>
JavaScript:
// Pak de titel
document.getElementById("titel")

// Maak hem rood
document.getElementById("titel").style.color = "red";
Let op: Je id in HTML moet exact hetzelfde zijn als in JavaScript! (hoofdletters tellen)

Functies met onclick (knop reageert)

HTML:
<button onclick="zegHallo()">
  Klik mij
</button>
JavaScript:
function zegHallo() {
  alert("Je klikte!");
}
Wat gebeurt er?
  1. Gebruiker klikt op knop
  2. onclick roept functie zegHallo() aan
  3. Alert verschijnt

Tekst veranderen met innerHTML

HTML:
<p id="tekst">Oude tekst</p>
<button onclick="verander()">
  Verander tekst
</button>
JavaScript:
function verander() {
  document.getElementById("tekst").innerHTML = "Nieuwe tekst!";
}

Stijl aanpassen met style

Kleur veranderen:

function maakRood() {
  document.getElementById("tekst").style.color = "red";
}

Element verbergen/tonen:

function verberg() {
  document.getElementById("foto").style.display = "none";
}

function toon() {
  document.getElementById("foto").style.display = "flex";
}
Veel gebruikte style properties:
  • style.color → tekstkleur
  • style.backgroundColor → achtergrondkleur
  • style.display → verbergen/tonen ("none" of "flex"/"block")
  • style.fontSize → lettergrootte (bijv. "20px")

Dark Mode Toggle (voorbeeld)

HTML:
<div id="pagina">
  <h1>Mijn website</h1>
  <img id="zon" src="zon.png">
  <img id="maan" src="maan.png"
       style="display: none;">
  <button onclick="darkMode()">
    Toggle Dark Mode
  </button>
</div>
JavaScript:
function darkMode() {
  // Pagina donker maken
  let pagina = document.getElementById("pagina");
  pagina.style.backgroundColor = "black";
  pagina.style.color = "white";
  
  // Zon verbergen, maan tonen
  document.getElementById("zon").style.display = "none";
  document.getElementById("maan").style.display = "flex";
}

🔥 Veelgemaakte fouten

function zegHallo → function zegHallo() (haakjes!)
onclick="zegHallo" → onclick="zegHallo()" (haakjes!)
style.background-color → style.backgroundColor (camelCase)
Functienaam in HTML verschilt van JS → werkt niet!

Debug tips

Test altijd in de console:
function test() {
  console.log("Functie werkt!");
}
  • Zie je dit in de console? → Functie werkt
  • Zie je niks? → Check of onclick goed geschreven is
  • Check of je id's exact hetzelfde zijn!

Stappenplan Dark Mode

  1. Maak functie darkMode()
  2. Selecteer element met getElementById
  3. Verander style.backgroundColor en style.color
  4. Verberg/toon iconen met style.display
  5. Koppel functie aan knop met onclick
