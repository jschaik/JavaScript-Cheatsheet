Week 2 - Functies en onclick

Functies (herbruikbare code)

Wat is een functie?
Een functie is een stukje code dat je kunt hergebruiken. Je schrijft het √©√©n keer en gebruikt het zo vaak je wilt of pas wanneer je het nodig hebt.

Functie maken:

function zegHallo() {
  console.log("Hallo!");
}

Functie aanroepen:

zegHallo();
// Nu gebeurt het pas!
Onthoud: Functie maken = code opschrijven, aanroepen = code pas gebruiken

Functies met onclick (knop reageert)

HTML:
<button onclick="zegHallo()">
  Klik mij
</button>
JavaScript:
function zegHallo() {
  alert("Je klikte!");
}
Wat gebeurt er?
  1. Gebruiker klikt op knop
  2. onclick roept functie zegHallo() aan
  3. Alert verschijnt

Tekst veranderen met innerHTML

HTML:
<p id="tekst">Oude tekst</p>
<button onclick="verander()">
  Verander tekst
</button>
JavaScript:
function verander() {
  document.getElementById("tekst").innerHTML = "Nieuwe tekst!";
}

Stijl aanpassen met style

Kleur veranderen:

function maakRood() {
  document.getElementById("tekst").style.color = "red";
}

Element verbergen/tonen:

function verberg() {
  document.getElementById("foto").style.display = "none";
}

function toon() {
  document.getElementById("foto").style.display = "flex";
}
Veel gebruikte style properties:
  • style.color ‚Üí tekstkleur
  • style.backgroundColor ‚Üí achtergrondkleur
  • style.display ‚Üí verbergen/tonen ("none" of "flex"/"block")
  • style.fontSize ‚Üí lettergrootte (bijv. "20px")

Debug tips

Test altijd in de console:
function test() {
  console.log("Functie werkt!");
}
  • Zie je dit in de console? ‚Üí Functie werkt
  • Zie je niks? ‚Üí Check of onclick goed geschreven is
